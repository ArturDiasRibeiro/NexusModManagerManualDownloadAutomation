# NexusModManagerManualDownloadAutomation
*NO LONGER FUNCTIONAL* Automação híbrida (Python + Tampermonkey) para download de coleções no Vortex (Nexus Mods). Desenvolvida para contornar cliques manuais em contas Free, com suporte a monitores Ultrawide (5120x1440) via reconhecimento de imagem (OpenCV) e ROI centralizado, garantindo precisão em modais e fechamento automático de abas.
